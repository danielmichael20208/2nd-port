🔐 Password Strength Analyzer

A command-line cybersecurity tool that scores password strength using Shannon entropy calculations, multi-factor character-set analysis, and an offline common-password dictionary. It also performs live breach detection via the Have I Been Pwned API using k-anonymity — meaning your real password is never transmitted.

  • Shannon entropy calculation (bits) for accurate strength measurement
  • Crack-time estimation for both offline GPU attacks and online throttled logins
  • HIBP Pwned Passwords API — k-anonymity breach detection
  • AI-style improvement suggestions and passphrase recommendations
  • Structured JSON log events → auto-uploaded to SIEM dashboard

🛰️ Network Port Scanner

A Python TCP port scanner built from raw sockets. Goes beyond simple open/closed detection by providing full port state enumeration, service banner grabbing, TTL extraction, and basic OS fingerprinting — the same workflow used by real network security engineers.

  • Full state enumeration: open · closed · refused · filtered · timeout · error
  • Banner grabbing for service identification (SSH, HTTP, SMB, RDP)
  • TTL-based OS fingerprinting (Linux vs Windows vs Cisco)
  • Configurable port list and scan timeout
  • Structured JSON events auto-pushed to SIEM pipeline
Python 3 socket TCP Connect Banner Grab OS Fingerprint JSON Logging
SIEM Integration
  • Emits PORT_OPEN (WARN), PORT_CLOSED (INFO)
  • Emits PORT_FILTERED / PORT_REFUSED (WARN)
  • Context includes: host, port, state, service, TTL, OS guess, banner

Arduino · Embedded · Physical Security

🔒 CyberDoor v1 — Arduino Access Control

A hardware security project simulating a real access control system. Users enter a PIN on a 4×4 keypad; the system verifies it, controls a servo lock, drives LED indicators, and sounds an alarm on failure. After 3 failed attempts, the system enters a security lockdown mode and logs every event with timestamps and severity levels — matching the same format as the SIEM pipeline.

  • Keypad PIN authentication with configurable correct PIN
  • Servo motor lock control (0° = locked, 90° = unlocked)
  • Red/green LED indicators + piezo buzzer alarm
  • 3-strike lockdown with visual + audio alert sequence
  • Structured serial logging: [BOOT] [INFO] [WARN] [ALERT] [ACCESS]
  • Simulated in TinkerCAD Circuits; designed for real Arduino Uno

📊 DanielOS Cyber SIEM Dashboard

A live security analytics dashboard powered by JSON event logs from the password analyzer, port scanner, and CyberDoor tools. Includes Chart.js visualizations, a security score engine, alert detection rules, time-based filtering, keyword search, log export, and an auto-refresh toggle — all without a backend server.

  • Security score (0–100) calculated from event severity weights
  • Charts: events by level (bar), events by source (pie), timeline (line)
  • Detection rules: port scanning activity, brute force, weak credentials
  • CLI-style filters: logs source=scan level=warn since=2h
  • Export to CSV and JSONL formats directly in browser
  • GitHub API auto-upload pipeline from local Python tools
