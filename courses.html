Foundation Courses — Free

Build the bedrock: networking, Linux, and core security concepts. Every course listed here is 100% free with a certificate included.

🖧
Cisco NetAcad — Networking Basics

IP addressing, TCP/IP stack, routing fundamentals, DNS, DHCP, subnetting, and the OSI model. Essential before any hacking course.

FREE 10–15 hrs cisco.com/go/netacad In Progress
🔐
Cisco NetAcad — Introduction to Cybersecurity

Threat types, the CIA triad, SOC overview, malware categories, and real-world attack case studies. Industry-recognized certificate.

FREE 6–10 hrs Cisco Badge In Progress
🐧
TryHackMe — Pre-Security Learning Path

Web basics, Linux fundamentals, networking intro, and how the internet works — all in guided, gamified labs. Best starting point for beginners.

FREE TIER tryhackme.com ~20 hrs In Progress
🎓
IBM SkillsBuild — Cybersecurity Fundamentals

Malware analysis, access control, incident response basics, forensics intro, and enterprise defense concepts. Corporate-recognized credential.

FREE IBM Badge ibm.com/training Planned
🔑
SANS Cyber Aces — Online Tutorials

SANS Institute's free intro course covering OS internals, networking, and systems administration from a security perspective.

FREE cyberaces.org Planned
02 // stage 2

Tools & Hands-On Labs

Move from theory to action. These platforms put you inside real Red Team and Blue Team workflows.

Red Team (Offensive)

💀
OverTheWire — Bandit (Wargames)

The best free Linux privilege escalation wargame. 34 levels of increasing difficulty teaching SSH, file permissions, grep, find, and scripting under pressure.

FREE overthewire.org Linux / SSH In Progress
🏴
PicoCTF — Capture the Flag

Carnegie Mellon's free CTF platform. Covers web exploitation, binary exploitation, forensics, reverse engineering, and cryptography in game-like challenges.

FREE picoctf.org All Domains In Progress
🎯
TryHackMe — Jr Penetration Tester Path

Structured offensive security path: reconnaissance, exploitation, web app attacks, Metasploit, privilege escalation, and post-exploitation. Real hacking methodology.

FREE TIER ~60 hrs Practical Labs Planned
📦
HackTheBox Academy — Free Tier

Module-based offensive security training. Free tier covers network fundamentals, web app basics, and introductory exploitation. Used by real penetration testers.

FREE TIER academy.hackthebox.com Planned
🕸️
OWASP WebGoat — Web App Vulnerability Lab

A deliberately insecure web application from the Open Web Application Security Project. Practice SQL injection, XSS, CSRF, and the full OWASP Top 10 safely.

FREE Local Install Web Security Planned

Blue Team (Defensive / SOC)

🛡️
CyberDefenders — Blue Team Labs

Realistic SOC analyst labs: log analysis, network forensics, malware triage, and incident investigation. Closest free platform to real SOC work.

FREE TIER cyberdefenders.org SOC / DFIR Planned
📡
Blue Team Labs Online — Free Tier

Defensive security labs focusing on threat hunting, log analysis, and incident response. Good complement to offensive TryHackMe paths.

FREE TIER blueteamlabs.online Planned
📊
Splunk Free Fundamentals 1

Learn the industry's most-used SIEM tool. Data ingestion, search, dashboards, and alerts. Free training from Splunk itself — shows up on every SOC job listing.

FREE education.splunk.com SIEM Planned
🔬
Wireshark — Official Documentation & Labs

Learn deep packet inspection using the industry-standard tool. Follow official Wireshark labs: HTTP captures, TCP analysis, DNS traffic, and suspicious activity detection.

FREE wireshark.org Already Installed In Progress

Certification Pipeline

All study materials are free. Exam fees are paid but affordable — many have scholarship programs for students.

Stage 1 — Foundation Certs

  • CompTIA Network+ (Study: Professor Messer — free)
  • Linux Essentials LPIC-1 (Free prep via NDG/NetAcad)
  • Google Cybersecurity Certificate (Coursera financial aid)
  • Cisco CyberOps Associate (Free training on NetAcad)
In Progress

Stage 2 — Security Core

  • CompTIA Security+ (Free prep: Professor Messer)
  • eJPT — Jr Penetration Tester (INE free learning path)
  • Microsoft SC-900 (Free training on Microsoft Learn)
  • Splunk Core User (Free Splunk training portal)
Planned

Stage 3 — Specialization

  • CCNA CyberOps (Cisco)
  • eWPT — Web App Pentesting (INE)
  • OSCP (Offensive Security — stretch goal)
  • GSEC / GCIA (SANS — long-term)
